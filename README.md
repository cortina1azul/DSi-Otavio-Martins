# Tarefa de Desenvolvimento de Sistemas — Java

## 1. Como configurar o Java no computador

Para programar em Java, é necessário instalar o **JDK (Java Development Kit)**, que contém as ferramentas necessárias para compilar e executar programas.

### Passo a passo

1. Acesse o site oficial da Oracle:
   https://www.oracle.com/java/technologies/downloads/

2. Baixe a versão mais recente do **JDK** compatível com seu sistema operacional (Windows, Linux ou Mac).

3. Execute o instalador e siga os passos da instalação.

4. Após instalar, é recomendado configurar a **variável de ambiente**.

2. Clique em **Editar variáveis de ambiente do sistema**.
